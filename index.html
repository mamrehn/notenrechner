Notenskala

Punktebereich Note

Nutzungshinweise:

Sie können die Notenskala anpassen und eine bestimmte Punktzahl hervorheben, indem Sie Parameter zur URL hinzufügen.

Parameter:

  • ?max={maximale_punktzahl}: Legt die maximale Punktzahl für die Prüfung/Aufgabe fest. Standard ist 100.
  • ?p={ihre_erreichten_punkte}: Hebt die Zeile hervor, die Ihren erreichten Punkten entspricht, und zeigt Ihre Note an.

Beispiele:

  • (11 Punkte von 13)
  • (85 Punkte von 100)
